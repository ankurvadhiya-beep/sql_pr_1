  Email        VARCHAR(100)   Customer email
  Address      VARCHAR(200)   Customer address

### **2. Orders Table**

Stores customer orders.

  Column        Type            Description
  ------------- --------------- ----------------------
  OrderID       INT, PK         Auto-incremented ID
  CustomerID    INT, FK         References Customers
  OrderDate     DATE            Order date
  TotalAmount   DECIMAL(10,2)   Order total

### **3. Products Table**

Stores product catalog.

  Column        Type            Description
  ------------- --------------- ---------------------
  ProductID     INT, PK         Auto-incremented ID
  ProductName   VARCHAR(100)    Product name
  Price         DECIMAL(10,2)   Product price
  Stock         INT             Stock quantity

### **4. OrderDetails Table**

Stores product-wise breakdown of each order.

  Column          Type            Description
  --------------- --------------- ---------------------
  OrderDetailID   INT, PK         Auto-incremented ID
